RULES

    We are seeking solutions capable of predicting the bitrate complexity of video segments with high accuracy and greater speed than the libx264 medium preset. Participants are required to utilize the training set (can be downloaded from here) using the latest version of FFMPEG, which can be downloaded from here. The FFMPEG configuration should include the libx264 codec with a medium preset and a constant rate factor (CRF) of 26. Participants are encouraged to employ any effective methods to exclude segments with scene cuts, as the test set comprises only segments from the same scene.

Evaluation

    We will assess your solutions based on the accuracy of predicting the bitrate (P) for the test set and the prediction speed (s). Pearson Correlation (P) will be employed to measure accuracy. The winners will be ranked using the ratio Q = α×P + β1⁄S α and β will be realeased soon.
